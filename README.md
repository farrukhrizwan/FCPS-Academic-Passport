# FCPS Academic Passport 📘
**A Resilient Excellence Framework for FC Public School and College, Dera Bugti.**

The **FCPS Academic Passport** is a Progressive Web App (PWA) designed to bridge the foundational literacy gap for students from PG to Grade 10. In response to the unique challenges of the Dera Bugti region, this tool provides offline-first access to 1,000 essential high-frequency English words.

## 🚀 Vision
To stabilize academic outcomes in high-turnover environments by empowering students and teachers with a standardized, mobile-first instructional framework.

## 🛠 Features
- **Offline Capable:** Works without internet in remote areas via Service Worker.
- **Mobile-First Design:** Optimized for low-end smartphones and student use.
- **Searchable Database:** 1,000+ words categorized by grade-specific "Survival Lists."
- **PWA Ready:** Installable directly on Android and iOS devices.
- **Academic Stabilization:** Built to ensure curriculum continuity despite staffing challenges.

## 📁 Repository Structure
- `index.html`: The core application, UI, and vocabulary database.
- `manifest.json`: PWA metadata and app identity.
- `sw.js`: Service Worker script for offline caching.
- `logo.png`: School institutional branding.

## 📋 Implementation for Staff
1. **5-a-Day Method:** Teachers are encouraged to introduce 5 words daily from the grade list.
2. **IT Hub Backup:** Used as a digital instructional proxy during staff transitions.
3. **Assessment:** Weekly vocabulary "Spelling Bee" challenges based on the app's database.
